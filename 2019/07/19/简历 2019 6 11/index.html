项目经历

项目一 ： 批单服务

  • 软件架构 : SpringBoot + SpringCloud + SpringMVC + MyBatis

  • 开发环境 : IDEA + Maven + MySql + GitLab

  • 项目描述 :

    批单服务是在保险公司出单成功后，客户想要对保单进行修改，如修改被保人信息、投保人信息、投保金额附加险等信息。批单服务是为各个保险渠道提供服务的，为了提高服务的可扩展性，采用了一种报文机制。根据不同渠道的批改需求配置不同的报文。

  • 职责描述 :

  1. 配置批单批改报文（为了减轻工作量，用JavaSwing开发了一个报文配置工具）
  2. 开发批单数据迁移的接口（将老系统批单统一迁移到新系统中）。
  3. 开发教育渠道和医疗渠道特有的一些批单服务接口。
  4. 配合医疗和教育渠道联调批单报文。
项目二 : 清单解析服务

  • 软件架构 : SpringBoot + SpringCloud + SpringMVC + Mybaits

  • 开发环境 : IDEA + Maven + MySql + GitLab

  • 项目描述 :

    清单解析服务是为所有渠道提供清单解析、校验、解析成功后结果的保存等服务。需要根据清单模板配置相应的解析规则。解析规则包含表级的规则（如人数统计）、行级的规则（如根据证件类型对证件号的合法性进行校验）、列级规则（如某列的取值范围）。

  • 责任描述 :

  1. 参与数据库表的设计和修改。
  2. 在原有框架基础上进行项目的搭建。
  3. 开发基础的清单校验规则和解析结果的入库逻辑。
项目三 : 业务管理系统（老系统新功能模块开发）

  • 软件架构 : SpringMVC + ibatis + JSP

  • 开发环境 : Eclipse + SVN + JDK1.6 + Tomcat7 + Oracle

  • 项目描述 :

    该项目是整个公司的业务管理系统。管理总公司和分公司的业务。项目的页面是用JSP写的，后台采用了SpringMVC + ibatis + Oracle 架构。采用DWR框架从服务器请求数据。

  • 责任描述 :

  1. 负责分出分入指标管理模块的开发（包括JSP界面和接口）。
项目四 : 家庭网盘（毕业设计项目）

  • 软件架构 : SpringBoot + SpringMVC + MyBatis + LayUI

  • 开发环境 : IDEA + Maven + Mysql

  • 项目描述 :

    毕业设计项目选题，模拟百度网盘，实现文件的上传下载，媒体文件的预览。文件存储采用FastDFS文件存储系统。后台采用SpringBoot + SpringMVC + Mybatis架构。采用前后分离的方式，前端使用LayUI框架。采用Thymeleaf模板引擎生成页面。使用Shiro实现登陆和角色权限的校验。页面的菜单可以通过后台管理界面进行配置，实现对角色权限的控制。
